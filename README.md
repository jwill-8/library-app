# Library App

A simple vanilla JavaScript library tracker where you can add books, toggle read status, and remove books.

## Live Site

Current working site: https://jwill-8.github.io/odinProject/JS/Library/

## Features

- Add books with title, author, pages, and read state
- Toggle each book between read/unread
- Remove books from the list
- Book list persists in `localStorage`

## Run Locally

Open `index.html` in your browser.
